# AI Pathfinding Visualizer (Python + Pygame)

An interactive AI pathfinding simulator that demonstrates classical search algorithms using real-time visualization.

## Features

* BFS (Breadth First Search)
* DFS (Depth First Search)
* UCS (Uniform Cost Search)
* DLS (Depth Limited Search)
* IDDFS
* Bidirectional Search
* Dynamic obstacles
* Adjustable speed
* Interactive grid

## Technologies Used

* Python
* Pygame

## Installation

1. Clone the repository:

```
git clone https://github.com/your-username/ai-pathfinder.git
```

2. Install dependencies:

```
pip install -r requirements.txt
```

3. Run the project:

```
python main.py
```

## Controls

* Left click: draw walls
* Right click: remove walls
* Drag start/target nodes
* Choose algorithm from UI panel
